## Upgrade Guide

### From v2.0.x to v2.1.0

<Steps>
  <Step title="Backup Your Data">
    ```bash
    # Backup your results and configuration
    cp -r ~/mcpbench-results ~/mcpbench-results-backup
    cp ~/.config/mcpbench/config.yaml ~/.config/mcpbench/config.yaml.backup
    ```
  </Step>
  <Step title="Update MCPBench">
    ```bash
    # Update to latest version
    pip install --upgrade mcpbench
    ```
  </Step>
  <Step title="Update Configuration">
    Update your configuration file to use new options and features.
  </Step>
  <Step title="Verify Installation">
    ```bash
    # Test the installation
    mcpbench test-installation
    ```
  </Step>
</Steps>

## Migration Guide

### Result Format Changes

**Old Format (v2.0.x)**:
```json
{
  "task_id": "build-linux-kernel",
  "success": true,
  "time": 1847.3
}
```

**New Format (v2.1.0)**:
```json
{
  "task_id": "build-linux-kernel",
  "status": "success",
  "execution_time": 1847.3,
  "start_time": "2024-12-15T10:30:00Z",
  "end_time": "2024-12-15T11:00:47Z",
  "metrics": {
    "cpu_usage": 0.75,
    "memory_usage": 1024,
    "disk_io": 512
  }
}
```

### Configuration Migration

Use our migration tool to automatically update your configuration:

```bash
# Run the migration tool
mcpbench migrate-config

# Or manually update your config file
mcpbench update-config --from-version 2.0.5
```

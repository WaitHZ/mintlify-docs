# Crack 7z Archive

Security challenge involving password cracking and file extraction from encrypted archives.

## Task Description

You need to create a file called "solution.txt" with the word found "secret_file.txt" in the "secrets.7z" archive. The final "solution.txt" should be located at "/app/solution.txt".

## Requirements

- Crack the password for the 7z archive
- Extract the secret file
- Create solution.txt with the found word

## Expected Output

- File: `/app/solution.txt` containing the secret word

## Difficulty

**Medium** - Requires knowledge of password cracking techniques

# Installation Guide

This guide provides detailed instructions for installing MCPBench and setting up your evaluation environment.

## System Requirements

### Minimum Requirements

- **OS**: Linux (Ubuntu 20.04+ recommended)
- **RAM**: 8GB minimum, 16GB recommended
- **Storage**: 20GB free space
- **CPU**: 4 cores minimum, 8 cores recommended
- **Network**: Internet connection for downloading tasks and dependencies

### Recommended Environment

- **OS**: Ubuntu 22.04 LTS
- **RAM**: 32GB
- **Storage**: 100GB SSD
- **CPU**: 16 cores
- **GPU**: Optional, for certain evaluation tasks

## Installation Methods

### Method 1: Quick Install (Recommended)

```bash
# Download and run the installation script
curl -fsSL https://install.mcpbench.ai | bash

# Verify installation
mcpbench --version
```

### Method 2: Manual Installation

<Steps>
  <Step title="Clone Repository">
    ```bash
    git clone https://github.com/mcpbench/mcpbench.git
    cd mcpbench
    ```
  </Step>
  <Step title="Install System Dependencies">
    ```bash
    # Ubuntu/Debian
    sudo apt-get update
    sudo apt-get install -y build-essential curl wget git python3 python3-pip
    
    # CentOS/RHEL
    sudo yum update
    sudo yum install -y gcc gcc-c++ curl wget git python3 python3-pip
    ```
  </Step>
  <Step title="Install Python Dependencies">
    ```bash
    pip3 install -r requirements.txt
    ```
  </Step>
  <Step title="Install MCPBench">
    ```bash
    pip3 install -e .
    ```
  </Step>
  <Step title="Verify Installation">
    ```bash
    mcpbench test-installation
    ```
  </Step>
</Steps>

### Method 3: Docker Installation

```bash
# Pull the official Docker image
docker pull mcpbench/mcpbench:latest

# Run MCPBench in a container
docker run -it --rm mcpbench/mcpbench:latest
```

    ```bash
    pip install mcpbench
    ```
  </Accordion>
  <Accordion title="System-wide Installation">
    ```bash
    # Install system-wide (requires sudo)
    sudo pip3 install mcpbench
    ```
  </Accordion>
</AccordionGroup>

### Configuration

Create a configuration file:

```bash
# Create config directory
mkdir -p ~/.config/mcpbench

# Create configuration file
cat > ~/.config/mcpbench/config.yaml << 'EOF'
evaluation:
  timeout: 3600  # 1 hour timeout per task
  max_retries: 3
  parallel_tasks: 4

storage:
  results_dir: ~/mcpbench-results
  cache_dir: ~/.cache/mcpbench
  temp_dir: /tmp/mcpbench

logging:
  level: INFO
  file: ~/mcpbench.log
EOF
```


  <Accordion title="Permission Issues">

    ```bash
    # Fix permission issues
    ```

  </Accordion>
  <Accordion title="Python Version Issues">

  </Accordion>

## Advanced Configuration

### Custom Task Directories

```bash
# Set custom task directory
export MCPBENCH_TASK_DIR=/path/to/custom/tasks

# Or in config file
echo "tasks:
  custom_dir: /path/to/custom/tasks" >> ~/.config/mcpbench/config.yaml
```

### Performance Tuning

```yaml
# ~/.config/mcpbench/config.yaml
performance:
  max_workers: 8  # Number of parallel workers
  memory_limit: 16GB  # Memory limit per task
  cpu_limit: 4  # CPU cores per task
```

### Logging Configuration

```yaml
# ~/.config/mcpbench/config.yaml
logging:
  level: DEBUG
  format: "%(asctime)s - %(name)s - %(levelname)s - %(message)s"
  file: ~/mcpbench.log
  max_size: 100MB
  backup_count: 5
```

## Uninstallation

### Remove MCPBench

```bash
# Uninstall Python package
pip uninstall mcpbench

# Remove configuration files
rm -rf ~/.config/mcpbench
rm -rf ~/.cache/mcpbench

# Remove results (optional)
rm -rf ~/mcpbench-results
```

### Clean System Dependencies

```bash
# Remove system packages (if not needed elsewhere)
sudo apt-get remove build-essential curl wget git
```

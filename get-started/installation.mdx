# Installation Guide

This guide will help you set up the MCPBench evaluation environment on your system.

## System Requirements

### Minimum Requirements

- **Operating System**: Linux (Ubuntu 20.04+), macOS (10.15+), or Windows 10+
- **Python**: 3.8 or higher
- **Memory**: 4GB RAM minimum, 8GB recommended
- **Storage**: 10GB free space
- **Docker**: For containerized evaluation (recommended)

### Recommended Requirements

- **Operating System**: Ubuntu 22.04 LTS
- **Python**: 3.10 or higher
- **Memory**: 16GB RAM
- **Storage**: 50GB free space
- **GPU**: NVIDIA GPU with CUDA support (for ML tasks)

## Installation Methods

### Method 1: Docker Installation (Recommended)

Docker provides the most consistent evaluation environment across different systems.

#### 1. Install Docker

**Ubuntu/Debian:**
```bash
sudo apt update
sudo apt install docker.io docker-compose
sudo systemctl start docker
sudo systemctl enable docker
sudo usermod -aG docker $USER
```

**macOS:**
```bash
# Install Docker Desktop from https://www.docker.com/products/docker-desktop
```

**Windows:**
```bash
# Install Docker Desktop from https://www.docker.com/products/docker-desktop
```

#### 2. Pull MCPBench Image

```bash
docker pull mcpbench/evaluation:latest
```

#### 3. Run Evaluation Container

```bash
docker run -it --rm \
  -v $(pwd):/workspace \
  -v /var/run/docker.sock:/var/run/docker.sock \
  mcpbench/evaluation:latest
```

### Method 2: Local Installation

#### 1. Clone Repository

```bash
git clone https://github.com/mcpbench/mcpbench.git
cd mcpbench
```

#### 2. Create Virtual Environment

```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

#### 3. Install Dependencies

```bash
# Install core dependencies
pip install -r requirements.txt

# Install development dependencies (optional)
pip install -r requirements-dev.txt
```

#### 4. Install MCPBench

```bash
pip install -e .
```

## Environment Setup

### 1. Configure Environment Variables

Create a `.env` file in your project root:

```bash
# MCPBench Configuration
MCPBENCH_DATA_DIR=/path/to/data
MCPBENCH_CACHE_DIR=/path/to/cache
MCPBENCH_LOG_LEVEL=INFO

# Docker Configuration (if using Docker)
DOCKER_REGISTRY=mcpbench
DOCKER_IMAGE_TAG=latest

# Evaluation Configuration
EVALUATION_TIMEOUT=3600
MAX_CONCURRENT_EVALUATIONS=4
```

### 2. Initialize Data Directory

```bash
mcpbench init --data-dir /path/to/data
```

### 3. Download Task Data

```bash
mcpbench download-tasks --all
```

## Verification

### 1. Run System Check

```bash
mcpbench check-system
```

Expected output:
```
✓ Python version: 3.10.0
✓ Docker: Available
✓ Required packages: Installed
✓ Data directory: Configured
✓ Task data: Downloaded
```

### 2. Run Test Evaluation

```bash
mcpbench run --task hello-world --agent examples/simple-agent.py
```

## Troubleshooting

### Common Issues

#### Docker Permission Denied

```bash
# Add user to docker group
sudo usermod -aG docker $USER
# Log out and log back in
```

#### Python Version Issues

```bash
# Use pyenv to manage Python versions
curl https://pyenv.run | bash
pyenv install 3.10.0
pyenv global 3.10.0
```

#### Memory Issues

```bash
# Increase Docker memory limit
# In Docker Desktop: Settings > Resources > Memory
```
